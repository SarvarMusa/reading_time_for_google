# Reading Time for Google Chrome

Description: A simple and efficient Chrome extension that calculates and displays the estimated reading time for any article or webpage you're currently viewing. It solves the problem of not knowing how much time you need to invest in reading an article before you start.

## Tech Stack
- **Languages:** JavaScript, HTML, CSS
- **Platform:** Google Chrome Extensions API (Manifest V3)

## Setup instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/SarvarMusa/reading_time_for_google.git
   ```
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** in the top right corner.
4. Click on **Load unpacked** and select the `reading_time_for_google` directory.
5. The extension should now be installed and active in your browser.

## Environment variables
No environment variables are required for this project.

## Usage examples
1. Navigate to any article or blog post on the web.
2. The extension will automatically calculate the reading time based on the text content.
3. Look for the reading time indicator displayed on the page.
